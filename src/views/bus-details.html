Туристический автобус MAN Regio

2012 года выпуска с салоном на 59 пассажиров. В автобусе есть всё необходимое для максимального комфорта пассажиров: салон оборудован системой кондиционирования, полками для мелкого багажа, большим багажным отделением, кулер с горячей водой, микрофон, TV/DVD система, индивидуальные ремни безопасности. Идеален для перевозки деловых и туристических групп, школьников, и пассажирских перевозок по меж городу и за границу.

Оплатить заказ можно следующими способами:

  • Оплата через Сбербанк-онлайн:

    Для оплаты через Сбербанк-онлайн, Вам необходимо связаться с менеджерами компании и уточнить реквизиты счета, на который нужно перечислить необходимую сумму.

  • Оплата в офисе:

    Для оплаты в офисе нашей компании, Вам необходимо оставить заявку на почте: info@absolute-trans.ru, указав подробности заказа или позвонить по телефону +7(812)305-90-30, получить расчет, затем подъехать с паспортом по адресу Санкт-Петербург, Коломяжский пр-т д.18, оф. 3-055

    Часы работы офиса:

    Пн-Пт : с 9:00-20:00

    Сб-Вс : с 10:00-18:00

  • Оплата для юридических лиц:

    Для оплаты по безналичному расчету от юридического лица, необходимо оставить заявку в свободной форме на почте: info@absolute-trans.ru реквизитами компании, для оформления договора и выставления счета.

  • Оплата онлайн через яндекс. кассу

    заполните форму по ссылке и нажмите кнопку «Оплатить». Откроется страница сервиса Яндекс.Касса, где вы сможете выбрать удобный способ оплаты.
